# SimpleBlog

SimpleBlog — это простое блог-приложение на базе Ruby on Rails. Для получения зачета по парадигмам!!!

Установка

- ruby 3.3.5
- Rails 7.2.1
- node v20.17.0

1. Склонируйте репозиторий


```bash
git clone https://github.com/davasilenko/simple_blog.git
cd simple_blog
bundle install
rails db:create
rails db:migrate

# rails db:seed

rails server
